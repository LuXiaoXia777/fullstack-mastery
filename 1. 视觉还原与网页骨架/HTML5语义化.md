# 01. 构建有灵魂的网页骨架：HTML5 语义化
> "语义化不仅是给图层起名，更是为了让机器读懂设计的灵魂。"

### 💡 核心感悟 (Quotes)
在 UI 设计中，我们追求像素级还原；在全栈开发中，我们追求“结构级”表达。用正确的标签包裹内容，是让网页具备 SEO 和无障碍能力的基石。

### 🎯 学习要点
1. **摒弃 Div-Soup (标签汤)**：不再用无意义的 `div` 堆砌页面。
2. **布局组件化**：使用 `nav` 定义导航，`main` 定义主体，`article` 定义独立内容。
3. **设计映射**：将 Figma 的 Frame 逻辑直接转化为 HTML 结构。

### 🛠️ 知识点拆解
| UI 模块 (Figma) | HTML5 标签 | 全栈用途 |
| :--- | :--- | :--- |
| 导航栏 | `<nav>` | 告诉搜索引擎这是全局跳转入口 |
| 页面主内容 | `<main>` | 网页最核心、唯一的内容区域 |
| 作品列表项 | `<article>` | 独立、可复用的内容块 |
| 底部版权信息 | `<footer>` | 存放备案、社交链接等辅助信息 |
