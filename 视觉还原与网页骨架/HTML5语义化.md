### 1. 什么是语义化？ (设计师视角)
语义化就是给网页的“图层”起准确的名字。
*   以前：全部用 `div` (相当于 Figma 里所有的图层都叫 Group)。
*   现在：用 `nav`, `header`, `main`, `footer` (相当于 Figma 里清晰的命名)。

### 2. 设计稿与标签的对应关系
| UI 模块 (Figma) | HTML5 标签 | 用途 |
| :--- | :--- | :--- |
| 导航栏 | `<nav>` | 告诉浏览器这是菜单 |
| 页面主内容 | `<main>` | 网页最核心的部分 |
| 作品列表项 | `<article>` | 独立的内容块 |
| 底部版权信息 | `<footer>` | 页面最下方的信息 |
